TASK-1 :

Create a HTML page and show the procedure of connecting JS file with it.

ANS :

Connecting JS file with HTML file can be done in two ways :

INTERNAL LINKING

We can include the JavaScript code directly within HTML document itself.

Step 1: Add script tag inside HTML file in the end.

Step 2: Write JavaScript code inside script tag. 

eg:  
                    <script> 
                        var a = 10; 
                        console.log(a);
                    </script>

EXTERNAL LINKING

In External linking the JavaScript code is added in a new file with the extension .js

Step 1: Create a new file with the name index.js

Step 2: Link the JS file with the HTML file. 

eg: 
                    <script  src="index.js">

Step 3: Now, add JavaScript code inside index.js file. 

                    var a = "HELLO JS";
                    console.log(a);
